# Prospector's Instinct

Prospector's Instinct is an immersive client-side quality-of-life mod for Vintage Story.

Instead of revealing ore locations or replacing prospecting mechanics, the mod provides subtle particle guidance while holding a Prospecting Pick, helping players develop a natural sense for nearby mineral deposits.

The goal is to enhance exploration without removing the satisfaction of discovery.

## Features

- Particle guidance toward nearby ore
- Requires a Prospecting Pick
- Terrain-aware particle placement
- Located Mode when standing above ore
- Multiplayer compatible
- Lightweight and performance friendly
- Fully configurable

## Requirements

Vintage Story 1.22.3
